## 🧠 Conceitos Aprendidos

### O que é EC2?
Servidores virtuais na nuvem AWS, escaláveis e pagos por uso.

### Security Group
Firewall virtual que controla o tráfego de entrada e saída da instância.

### EBS (Elastic Block Store)
Armazenamento persistente anexado à instância EC2.

### Snapshot
Backup do volume EBS armazenado no S3.
